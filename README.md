SM Enterprises - E-Commerce Platform
A professional, full-stack e-commerce web application designed specifically for SM Enterprises to sell customized acrylic trophies and nameplates online.

🚀 Features
Dynamic Product Catalog: Browse a variety of acrylic products with real-time updates from the database.

Customization Engine: Allows users to provide specific text and design instructions for their trophies.

Order Management: A streamlined process for users to place orders and for admins to track them.

Responsive Design: Fully optimized for both desktop and mobile viewing.

Secure Backend: Built with robust logic to handle product data and customer inquiries.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript

Backend: PHP

Database: SQL (MySQL)

📁 Project Structure
Plaintext
├── assets/             # Images, CSS, and JS files
├── config/             # Database connection settings
├── includes/           # Reusable PHP components (header, footer)
├── admin/              # Dashboard for managing products and orders
└── index.php           # Main landing page
⚙️ Installation & Setup
Clone the repository:

Bash
git clone https://github.com/jadhavsudesh531-coder/SM-ENTERPRISES.git
Database Setup:

Import the provided .sql file into your local MySQL server (e.g., XAMPP/WAMP).

Update config/db.php with your local credentials.

Run the project:

Move the folder to your htdocs directory.

Access it via localhost/SM-ENTERPRISES.
